# 🎨 魔法填色画板 - 儿童互动绘画

专为儿童设计的Web端互动填色画板，让孩子轻松且充满成就感地完成绘画作品！

## ✨ 特色功能

### 🖌️ 拟真画笔体验
- 圆形笔触，模拟真实画笔效果
- 平滑的轨迹跟踪，支持鼠标和触摸屏操作
- 画笔大小适中，适合儿童手指操作

### 🛡️ 智能防溢出技术
采用**双层Canvas结构 + 像素掩码技术**：
- **底层（Color Layer）**：绘制彩色内容
- **顶层（Line Art Layer）**：显示黑白线稿，始终在最上方
- **像素级检测**：颜色只能在线框内部绘制，绝不会画到外面

### 🌈 丰富的调色板
- 12种鲜艳的纯色：红、黄、蓝、绿、紫、橙等
- 橡皮擦工具，支持擦除已涂的颜色
- 一键清空功能，快速重新开始

### 🎯 精心设计的界面
- 渐变紫色背景，视觉舒适
- 圆角卡片设计，安全友好
- 弹跳动画标题，增添趣味性
- 响应式布局，支持平板和手机

## 🐱 可爱的小猫线稿

## 🚀 快速开始

### 方式一：直接打开
双击 `index.html` 文件即可在浏览器中打开。

### 方式二：使用本地服务器

如果你想在特定端口运行（如2345端口）：

```bash
# 使用 Python
python3 -m http.server 2345

# 或使用 Node.js (需要安装 http-server)
npx http-server -p 2345

# 或使用 PHP
php -S localhost:2345
```

然后在浏览器中访问：`http://localhost:2345`

## 🎮 使用方法

1. **选择颜色**：点击底部调色盘中的颜色
2. **开始涂色**：按住鼠标左键（或触摸屏滑动）在线稿区域移动
3. **防溢出保障**：颜色会自动限制在线框内部，不用担心画到外面
4. **橡皮擦**：点击橡皮擦图标，可以擦除已涂的颜色
5. **清空重画**：点击"清空颜色"按钮，可以重新开始

## 🛠️ 技术实现

### 防溢出核心算法
```javascript
// 1. 创建掩码Canvas，存储线稿像素信息
// 2. 使用Flood Fill算法识别：
//    - 哪些区域是"外部"（不可涂色）
//    - 哪些区域是"内部"（可涂色）
// 3. 绘制时逐像素检测，只在可涂色区域绘制
```

### 双层Canvas架构
```
┌─────────────────────────┐
│  Line Art Layer (z=2)   │ ← 黑色线稿，始终可见
│  pointer-events: none    │
├─────────────────────────┤
│  Color Layer (z=1)      │ ← 用户绘制的颜色
│  接收鼠标/触摸事件        │
└─────────────────────────┘
```

## 📝 开发说明

如果你想修改线稿或添加新功能：

- **修改线稿**：编辑 `script.js` 中的 `drawLineArt()` 方法
- **添加颜色**：在 `index.html` 的调色板区域添加新的 `.color-option`
- **调整画笔大小**：修改 `brushSize` 属性（默认20像素）
